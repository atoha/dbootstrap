Dojo Bootstrap Theme A Dojo theme based on the awesome Bootstrap framework.

This theme is still under active development so some widgets have not yet been themed. Also, it has only been tested on Firefox 15 (Mac). Contributions and testing welcome!

Hi there! This is a live preview of the new Dojo Bootstrap theme. The existing Dojo themes are great, but I wanted to try and bring the beauty of Twitter's Bootstrap to Dojo.

Utilising the ease of Stylus for managing the CSS and testing against the Dojo Theme Tester the project has really started to come together.

However, there is still lots to do (especially testing cross-browser/platform) so if you discover an issue please report it or even better fork the project and submit a pull request.
